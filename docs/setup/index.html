 Mac — how to run

  1. Click Download for Mac above
  2. Open your Downloads folder
  3. Double-click mac-setup-signed.command
  4. If macOS blocks it: right-click → Open → Open
  5. Watch the terminal — done in ~2 min

🖥 Windows — how to run

  1. Click Download for Windows above
  2. Open your Downloads folder
  3. Right-click windows-setup.ps1
  4. Select Run with PowerShell
  5. If prompted, choose Run anyway
After the script finishes: open Claude Code by typing claude in your terminal. You'll be prompted to authenticate Atlassian and Slack inside Claude — follow the on-screen instructions.
